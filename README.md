## 🎮 О игре

Мы делаем атмосферный симулятор жизни бездомного в реалистичном (но ужатом) районе.
*   **Жанр:** Survival / Simulation / Dark Humor
*   **Движок:** Python + Ursina Engine
*   **Стиль:** Low Poly / Реализм

### 🔥 Ключевые фишки:
1.  **Реальный район:** Карта основана на нашем дворе (подъезды, лавки, магазины).
2.  **Экономика хлама:** Сбор бутылок, сдача вторсырья, торговля.
3.  **Зачекушье:** Секретная пасхалка с жутким бэкстейджем мира.
4.  **Юмор:** Смешные диалоги с местными бабушками и охраной.

## 🚀 Как запустить игру

1.  Установите [Python 3.9+](https://www.python.org/)
2.  Установите библиотеку Ursina:
```bash
    pip install ursina
```
3.  Клонируйте репозиторий:
```bash
    git clone https://github.com/ТВОЙ_НИК/homeless-simulator.git
```
4.  Запустите игру:
```bash
    python src/main.py
```

## 🛠️ Для разработчиков

### Как внести вклад?
1.  Сделайте форк или ветку (`git checkout -b feature-name`).
2.  Внесите изменения.
3.  Проверьте, что игра запускается без ошибок.
4.  Сделайте коммит с понятным описанием.
5.  Отправьте пул-реквест (или просто push, если есть права).

### Правила коммитов
*   `feat:` новая функция (например, `feat: added bottle collection`)
*   `fix:` исправление бага (например, `fix: player stuck in wall`)
*   `docs:` обновление документации
*   `assets:` добавление графики/звуков
