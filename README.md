## 🎮 О игре

Мы делаем атмосферный симулятор жизни бездомного в реалистичном районе.
*   **Жанр:** Survival / Simulation / Dark Humor
*   **Движок:** Python + Ursina Engine
*   **Стиль:** Low Poly / Реализм

### 🔥 Ключевые фишки:
1.  **Реальный район:** Карта основана на нашем дворе (подъезды, лавки, магазины).
2.  **Экономика хлама:** Сбор бутылок, сдача вторсырья, торговля.
3.  **Зачекушье:** Секретная пасхалка с жутким бэкстейджем мира.
4.  **Юмор:** Смешные диалоги с местными бабушками и охраной.
